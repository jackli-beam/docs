## 完整字段映射

### 第2节：Firma（公司名称）

**PDF字段：** "Firma: [公司名称]"

**EVA字段：**

| EVA字段 | 内容 | 规则 |
|---------|------|------|
| **Firmierung** | 完整法定名称 | 从PDF精确复制，包括法律形式 |
| **Kurzfirmierung** | 简称 | 简单：与完整相同。复杂：缩写 |
| **Alphasort** | 搜索关键词 | 2个以上单词，无禁用首字 |
| **Schlagwort** | 其他关键词 | 类似于Alphasort |

<Warning>
**Alphasort禁用的首字：**
- ❌ Verwaltungsgesellschaft
- ❌ Beteiligungsgesellschaft
- ❌ Schifffahrtsgesellschaft

可在位置2+使用
</Warning>

---

### 第2节：Sitz（地址）

**PDF字段：** "Sitz: [地址]"

**EVA字段：**

| EVA字段 | 内容 | 汉堡规则 |
|---------|------|----------|
| **Straße** | 街道名称 | "Straße" → "Str." |
| **Hausnummer** | 门牌号 | 字母后缀小写："12a" |
| **PLZ** | 邮政编码 | 从下拉菜单选择（验证） |
| **Ort** | 城市 | 通常是"Hamburg" |
| **Land** | 国家 | 通常是"Deutschland" |

**示例：**
```
PDF："Hauptstraße 12A, 20095 Hamburg"
→ Straße："Hauptstr."
→ Hausnummer："12a"
→ PLZ："20095"（从下拉菜单选择）
→ Ort："Hamburg"
→ Land："Deutschland"
```

---

### 第4节：Geschäftsführer（董事）

**PDF字段：** "Geschäftsführer: [姓名、出生日期、地址]"

**EVA部分：** **Beteiligten**（每人多个子字段）

#### 姓名处理规则（关键！）：

<Tabs>
  <Tab title="德国居民">
    **如果居住地 = 德国城市：**

    **Vorname（名）：** 仅第一个名字

    **Schlagwort（关键词）：** 所有其他名字

    **示例：**
    ```
    PDF："Hans Peter Wilhelm Müller, wohnhaft in Hamburg"
    → Vorname："Hans"
    → Schlagwort："Peter Wilhelm"
    ```
  </Tab>
  <Tab title="外国居民">
    **如果居住地 = 外国城市：**

    **Vorname（名）：** 所有名字合在一起

    **Schlagwort（关键词）：** 留空或可选

    **示例：**
    ```
    PDF："Hans Peter Wilhelm Müller, wohnhaft in Wien, Österreich"
    → Vorname："Hans Peter Wilhelm"
    → Schlagwort：（留空）
    ```
  </Tab>
</Tabs>

---

### 第7节：Tag der Eintragung（注册日期）

**PDF字段：** "Tag der Eintragung: DD.MM.YYYY"

**EVA使用：** **关键 - 到处使用！**

此日期用作：
- 所有董事的**Beginndatum**
- 公司运营的开始日期
- 整个系统的参考日期

<Warning>
**对以下内容使用相同日期（Tag der Eintragung）：**
- 所有Beteiligten（董事）的Beginndatum
- 任何需要开始日期的其他日期字段
- **永远不要使用今天的日期或每位董事不同的日期！**
</Warning>

## 快速参考矩阵

| PDF位置 | PDF字段 | EVA部分 | EVA字段 | 特殊规则 |
|---------|---------|---------|---------|----------|
| **第2节** | Firma | Stammdaten | Firmierung、Kurzfirmierung、Alphasort | 精确复制，2个以上单词的Alphasort |
| **第2节** | Sitz | Anschrift | Straße、Hausnummer、PLZ、Ort、Land | 汉堡：小写后缀 |
| **第3节** | Stammkapital | Kapital | Stammkapital | 德国数字格式 |
| **第4节** | Geschäftsführer | Beteiligten | Nachname、Vorname、Schlagwort、出生日期、Funktion、Beginndatum | 姓名规则因居住地而异 |
| **第7节** | Tag der Eintragung | Beteiligten | Beginndatum（所有董事） | 所有人相同日期 |
