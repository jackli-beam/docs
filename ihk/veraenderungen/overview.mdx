## Veränderungen (Changes/Modifications)

This section covers processing **changes and modifications** (Ummeldungen/Veränderungen) to existing company registrations in the EVA system.

<Info>
**Documentation in progress**

Complete documentation for Veränderungen will be added soon.
</Info>

## Process Characteristics

<Note>
**Complexity:** ⭐⭐ Medium
**Average Time:** 20-30 minutes per entry
**Priority:** Process 3rd (last)
</Note>

## Key Differences from Neueintragungen

<CardGroup cols={2}>
  <Card title="Must Exist" icon="magnifying-glass">
    100% of entries must already exist in system
  </Card>
  <Card title="Updates Only" icon="pen">
    No new company creation
  </Card>
  <Card title="Partial Data" icon="file">
    Only changed fields in PDF
  </Card>
  <Card title="History Tracking" icon="clock-rotate-left">
    Must preserve historical data
  </Card>
</CardGroup>

## Common Change Types

- Address changes (Sitzverlegung)
- Director/management changes
- Legal form changes
- Capital changes
- Business purpose modifications
- Name changes (Umfirmierung)
