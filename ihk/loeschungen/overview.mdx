## Standard Löschung Process

Based on the workshop recording, here's the core workflow:

<Steps>
  <Step title="Find Existing Company">
    Search and locate the company to be deleted in EVA system
  </Step>
  <Step title="Verify Liquidation Status">
    Check that liquidation status in PDF (Section 6) matches system
  </Step>
  <Step title="Set Business Closure">
    Mark "endgültige Betriebsaufgabe" (final business closure)
  </Step>
  <Step title="Transfer Closure Date">
    Apply closure date to all associated persons/directors
  </Step>
  <Step title="Clear Person Records">
    Clear "Personen in Firma" section (with exceptions for certain registers)
  </Step>
  <Step title="Add Closure Text">
    Add closure text with date
  </Step>
  <Step title="Complete Entry">
    Verify status "vollständig" and mark as "bearbeitet"
  </Step>
</Steps>

## Key Requirements

<Warning>
**Critical Points for Löschungen:**

1. **Company must exist** - 100% must find existing entry
2. **Verify liquidation** - Section 6 status must match
3. **Date consistency** - Use Löschungsdatum for all fields
4. **Person handling** - Apply closure to all Beteiligten
5. **Register exceptions** - Some registers keep person data
</Warning>

## Process Details

### Finding the Company

- Search by company name and registration number
- Entry **must** already exist in system
- If not found, escalate (possible data issue)

### Setting Closure Status

- Mark "endgültige Betriebsaufgabe"
- Use Tag der Löschung from PDF Section 7
- Status affects all related records

### Person Management

- Transfer closure date to all persons
- Clear "Personen in Firma" section
- **Exception:** Certain register types retain person data

### Text Entry

- Add standard closure text
- Include Löschungsdatum
- Format: "[Company Name] - Löschung am [Date]"
