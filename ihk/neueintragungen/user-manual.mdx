## 8-Phase Processing Guide

Complete User Manual content is available in the source documentation. Below are the key phases:

### Phase 1: Analyze the PDF (1-2 min)

**Essential checks:**
- **Section 1:** Entry number (HRB/HRA)
- **Section 2:** Company name & location
- **Section 4:** Directors count
- **Section 7:** Registration date (Beginndatum)

**Red flags:**
- "Sitzverlegung" ‚Üí Search old AND new names
- "Zweigniederlassung" ‚Üí Branch, not HQ
- "eGbR" ‚Üí Process under HR Sonstiges

### Phase 2: Search for Existing Entry (3-5 min) ‚ö†Ô∏è CRITICAL

<Steps>
  <Step title="Configure Search">
    Open **"Allgemein" ‚Üí "Suche"**

    Select: ‚òë Firmen ‚òë Personen

    Mode: **"√Ñhnlich suchen"**
  </Step>

  <Step title="Search Company Name">
    Enter 2-3 meaningful words from company name

    ‚ùå Don't search "GmbH" or "Verwaltungsgesellschaft" alone
  </Step>

  <Step title="Analyze Results">
    - **No results** ‚Üí Proceed to create new
    - **Betriebsst√§tte found** ‚Üí Convert to Hauptsitz
    - **Externer Hauptsitz** ‚Üí Update & convert
    - **Exact match** ‚Üí May be duplicate, check supervisor
  </Step>
</Steps>

[View complete search workflow ‚Üí](/ihk/neueintragungen/user-manual#phase-2-search)

### Phase 3: Create New Entry (30 sec)

Only if Phase 2 found NO existing entries!

1. Right-click ‚Üí Select entry type:
   - **Hauptsitz** (90%+ cases - standard GmbH, UG, AG)
   - **Zweigniederlassung** (branches only)
   - **GBR** (eGbR partnerships)

### Phase 4: Fill Company Data (2-3 min)

1. Click **"Gewerbedaten f√ºllen"** (but verify everything!)
2. Enter **Firmierung** exactly from PDF
3. Enter **Alphasort** (2+ words, avoid prohibited first words)
4. Enter **Address** (Hamburg rules: lowercase suffixes like "12a")
5. Validate **PLZ** via dropdown

<Warning>
**Alphasort prohibited first words:**
‚ùå Verwaltungsgesellschaft
‚ùå Beteiligungsgesellschaft
‚ùå Schifffahrtsgesellschaft
</Warning>

### Phase 5: Enter Beteiligten (Directors) (3-5 min each)

<Note>
**Most complex phase** - follow carefully!
</Note>

**Step 1: Determine Function Code** üîë

<CardGroup cols={2}>
  <Card title="Code 24" icon="user">
    **Alleinvertretung**

    Single director OR each acts alone

    Keywords: "vertritt allein"
  </Card>
  <Card title="Code 8" icon="users">
    **Gemeinsame Vertretung**

    Must act together

    Keywords: "gemeinschaftlich", "durch zwei"
  </Card>
</CardGroup>

**Step 2: Search & Add Each Director**

1. Search by: Nachname + Vorname + Geburtsdatum
2. If found (DoB matches): Assign existing person
3. If not found: Create new person

**Step 3: Name Handling Rules** ‚ö†Ô∏è

<Tabs>
  <Tab title="German Residents">
    **Vorname:** First given name ONLY

    **Schlagwort:** Additional given names

    Example: "Hans Peter M√ºller, Hamburg"
    - Vorname: "Hans"
    - Schlagwort: "Peter"
  </Tab>
  <Tab title="Foreign Residents">
    **Vorname:** ALL given names together

    **Schlagwort:** Empty or optional

    Example: "Hans Peter M√ºller, Vienna"
    - Vorname: "Hans Peter"
  </Tab>
</Tabs>

**Step 4: Set Dates**

- **Beginndatum:** Tag der Eintragung (Section 7) - same for ALL directors

### Phase 6: Enter Branchenschl√ºssel (1-2 min)

1. Navigate to **"Branchenschl√ºssel"**
2. Analyze PDF Section 2 "Gegenstand" (Business Purpose)
3. Select 1-4 industry codes
4. Prioritize primary business activity

### Phase 7: Enter Capital (30 sec)

**If GmbH, UG, or AG:**

- Navigate to **"Stammkapital"**
- Copy amount from PDF Section 3
- Format: German numbers (25.000,00 EUR)

**Common amounts:**
- GmbH: 25.000,00 EUR minimum
- UG: 1,00 EUR minimum
- AG: 50.000,00 EUR minimum

### Phase 8: Review & Complete (1-2 min)

**Final checklist:**

<Steps>
  <Step title="Company Data">
    ‚úÖ Firmierung, Alphasort, Address complete
  </Step>
  <Step title="Directors">
    ‚úÖ All entered with correct Function Code

    ‚úÖ Beginndatum = Tag der Eintragung
  </Step>
  <Step title="Industry & Capital">
    ‚úÖ Branchenschl√ºssel (1-4 codes)

    ‚úÖ Capital amount (if applicable)
  </Step>
  <Step title="Save">
    Click **"Speichern"** or **"Fertigstellen"**
  </Step>
</Steps>

## Common Mistakes to Avoid

<AccordionGroup>
  <Accordion title="‚ùå Skipping Search Phase">
    **Consequence:** Duplicate entries, wasted time

    **Do:** ALWAYS search first - 50% already exist!
  </Accordion>

  <Accordion title="‚ùå Trusting Auto-Fill">
    **Consequence:** Wrong addresses, wrong names

    **Do:** Verify EVERY field against PDF
  </Accordion>

  <Accordion title="‚ùå Wrong Function Code">
    **Consequence:** Legal authority errors, liability issues

    **Do:** Read Section 4 carefully before entering directors
  </Accordion>

  <Accordion title="‚ùå Wrong Name Handling">
    **Consequence:** Data inconsistency, search problems

    **Do:** Check residence first - German vs Foreign rules differ
  </Accordion>

  <Accordion title="‚ùå Using Prohibited Alphasort Words">
    **Consequence:** System freeze, search failures

    **Do:** Use distinctive keywords, avoid common terms in position 1
  </Accordion>

  <Accordion title="‚ùå Wrong Beginndatum">
    **Consequence:** Timeline errors, audit failures

    **Do:** ALWAYS use Tag der Eintragung for all directors
  </Accordion>
</AccordionGroup>
